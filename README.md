Typical DMX lights and moving heads are large and expensive as they are meant for large venue shows. As someone getting into stage lighting, I wanted a practical way to practice and see my changes which I plan to do by building a suite of DMX fixtures starting with the most difficult which is the moving head. 
This system is small, making it perfect for table-top shows and low power. Uses an ESP32 with a library which allows DMX over ArtNet allowing me to wirelessly control functions via a DMX software such as QLC+
